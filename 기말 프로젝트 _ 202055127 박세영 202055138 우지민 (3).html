# 복제된 데이터를 기준으로 사망비율 비교
def graph_ratio(column):

    death_sum = data_new.groupby(column).danger.sum()

    death_count = data_new.groupby(column).danger.count()

    label = data_new.groupby(column).count().index
    
    N = len(data_new[column].unique())
    index = np.arange(N)
    
    #총 사망여부
    sum1= []

    for b in label : 
        sum1.append(death_sum[b])

    #총 사건 발생 수
    
    count = []

    for b in label : 
        count.append(death_count[b])

    #사망비율
    
    ratio= []

    for b in range(len(label)):
        ratio2 = sum1[b]/count[b]
        ratio.append(ratio2)
    
    plt.rcParams["figure.figsize"] = (14,4)
    
    plt.title(column+' & ratio', fontsize=30)

    plt.ylabel('death ratio', fontsize=18)

    plt.xlabel(column, fontsize=18)

    plt.xticks(index, label, fontsize=15)
    
    colors = sns.color_palette('hls',len(column))

    return plt.bar(label, ratio, width = 0.3,data = data_new,color=colors), ratio

3. 결론¶

3-1. 분석 결과¶

  • 독립변수가 (공사비, 공사종류, 공정율, 날씨, 계절) 종속변수에 미치는 영향은 대체로 미미함.
  • 가장 aic 가 낮았던 모델은 ? 공사비, 공사종류, 공정율, 날씨를 독립변수로 이용한 모델
  • 이를 통하여 예측 모델을 만든 결과, 54%의 낮은 정확도를 보임.
  • 사망의 예측 주요 원인 : 날씨
  • 사망의 실제 주요 원인 : 공사비
  • 월별, 장마철과 비장마철 등 다른 분류 기준으로 분석해보았지만, 사고 발생 시 사망자가 발생한 비율은 크게 차이가 없었음.

3-2. 한계 및 보완할 점¶

  • 위 예측 모델은 전제조건으로 '건설사고가 발생하였을 때'가 필요함.
  • 현재는 '건설 사고가 발생하였을 때, 사망 여부'를 예측하는 모델임. (조건부 확률)
  • 건설사고가 일어나지 않은 경우의 데이터를 입력하여 건설 사고가 발생하고 사망할 확률을 파악하는 모델로 보완해줄 필요가 있음.
  • 공정율과 공사비를 제외한 독립변수들은 범주형 데이터임.범주형 독립변수로 인해 항목이 세분화 되다 보니 각 계수의 유의미함이 떨어지게 나옴.
  • 공사비,공정율과 같은 수치형 독립변수가 많았다면 더 나은 예측모델이 나왔을 것으로 보임.

3-3. 시사점¶

  • 공사비가 높아질수록 사망자 비율이 줄어드는 경향을 파악할 수 있었음.
  • 아래 참고문헌 기사에는 공사비는 건설사고 발생과 큰 연관성이 없다 라고 밝힘.
  • 위 두 사실을 종합하면 공사비는 건설사고 발생 여부보다는 건설사고 발생 시, 사망 확률에 큰 영향을 미치므로 적정 공사비 확보가 중요하다 라고 생각됨.
